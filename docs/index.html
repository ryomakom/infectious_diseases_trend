都道府県別にみた主な感染症の流行状況

小宮山亮磨　朝日新聞デジタル企画報道部 — Home

国立健康危機管理研究機構の感染症発生動向調査の公開データをもとに、 都道府県別の主な感染症について、定点当たり患者数の週次推移を確認できます。 新型コロナウイルス感染症（COVID-19）、インフルエンザ、ＲＳウイルス感染症、感染性胃腸炎、手足口病、 水痘、突発性発しん、伝染性紅斑、ヘルパンギーナなどの流行状況を 地域ごとに比較できます。

グラフでは、都道府県や感染症を自由に追加して比較できます。 下部にあるスライダーを動かすと表示する期間を変えられます。左右のつまみを引っ張ると期間の長さも変わります。 GitHub Actionsを使ってデータを自動で収集し、D3.jsで可視化しています。
